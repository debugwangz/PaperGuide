# PaperGuide 产品路线图

## 愿景

让任何人都能读懂任何论文，无论背景如何。

## 核心问题

学术论文对新手不友好的原因：

1. **知识断层** - 论文假设读者已具备大量背景知识
2. **术语壁垒** - 专业术语没有解释
3. **逻辑跳跃** - 省略"显而易见"的推理步骤
4. **图表晦涩** - 图表缺乏足够解释

## 解决方案

PaperGuide 通过 AI 填补这些鸿沟：

- 自动检测论文中的"知识断层"
- 通过网络搜索补充背景知识
- 用对话方式解答疑惑
- 生成适合读者水平的解读文档

## 功能路线图

### Phase 1: MVP ✅

基础功能，已完成：

- [x] PDF 上传和解析
- [x] arXiv 链接支持
- [x] 论文分析生成 Review
- [x] 对话问答
- [x] 文档更新建议

### Phase 2: 知识增强 🚧

提升理解深度：

- [ ] 知识空白检测 - 自动识别论文中的知识断层
- [ ] 背景知识补充 - 调用搜索 API 获取解释
- [ ] 概念图谱 - 展示概念之间的关系
- [ ] 图表深度解读 - 详细解释每张图表

### Phase 3: 个性化 📋

根据用户背景定制：

- [ ] 用户画像 - 记住用户的知识水平
- [ ] 自适应解释 - 根据背景调整解释深度
- [ ] 学习路径 - 推荐需要先了解的前置知识
- [ ] 进度追踪 - 记录已理解的概念

### Phase 4: 协作与导出 📋

- [ ] 导出为 PDF
- [ ] 导出到 Notion
- [ ] 分享链接
- [ ] 团队协作批注

### Phase 5: 多模态增强 📋

- [ ] 论文朗读 (TTS)
- [ ] 视频讲解生成
- [ ] 公式手写识别
- [ ] 引用论文自动关联
