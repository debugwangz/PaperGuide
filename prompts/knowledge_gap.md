## 检测标准

1. **概念跳跃** - 是否有概念未经解释就使用？
2. **逻辑断层** - 推理过程是否有跳跃？
3. **依赖缺失** - 理解A是否需要先理解B，但B没有解释？
4. **深度不一** - 有些地方解释很详细，有些过于简略？

      "location": "问题出现的位置",

      "suggestion": "修复建议"
    }
  ]
}
