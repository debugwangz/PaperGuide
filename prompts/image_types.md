# 图片分类和解读提示词

分析这张论文中的图片，判断其类型并解读内容。

## 图片分类

请先判断图片属于以下哪种类型：

1. **架构图** - 模型结构、系统流程图
   - 重点解读：组件关系、数据流向、各模块功能

2. **流程图** - 算法步骤、训练流程
   - 重点解读：步骤顺序、条件分支、输入输出

3. **实验结果图** - 曲线图、柱状图、热力图
   - 重点解读：数值对比、趋势、得出的结论

4. **示意图** - 概念解释、直觉展示
   - 重点解读：核心思想的可视化表达

5. **其他** - 不属于以上类型
   - 自由解读图片内容
